Applet 有两类 API：

- Dart API：Flutter 宿主用来加载、限制、观察和重载 applet；
- JavaScript API：applet 模块用来描述 Flutter UI。

## 从哪里开始

| 页面 | 用途 |
| --- | --- |
| [Dart 宿主 API](/applet/zh/api/dart-host/) | `Applet.asset`、`Applet.source`、`AppletController`、Bundle 和运行时参数。 |
| [JavaScript API](/applet/zh/api/javascript/) | `State`、`Applet.action`、节点辅助方法、模块和默认导出。 |
| [组件模块](/applet/zh/api/components/) | `@app/material`、`@app/cupertino`、布局、widgets 和覆盖策略。 |

## 类型提示

支持 JavaScript 类型提示的编辑器可以引用 `types/app.d.ts`。它声明了内置模块、
常用属性、状态辅助函数、主题 token 和 Flutter 风格组件工厂。

## 兼容 alias

新代码应该导入 `@app/*` 模块。`@applet/*` 兼容 alias 仍然声明，以便旧脚本继续运行：

```js
import { Text } from "@app/material";
```

文档、示例和新 applet 都优先使用新的模块名。
